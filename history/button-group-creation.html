Using Bootstrap 5.1, create a HTML page that has a left-hand side button group containing for four services, ChatGPT, FOLIO, Airflow, and Github

    <div class="row">
      <div class="col-md-3">
        <div class="btn-group-vertical w-100">
          <button type="button" class="btn btn-primary">ChatGPT</button>
          <button type="button" class="btn btn-primary">FOLIO</button>
          <button type="button" class="btn btn-primary">Airflow</button>
          <button type="button" class="btn btn-primary">Github</button>
        </div>
      </div>
      <div class="col-md-9">
        <!-- Your main content here -->
      </div>
    </div>
  </div>
</body>
</html>

In this code, the left-hand side button group is created using the btn-group-vertical class to stack the buttons vertically. The w-100 class is used to make the button group take up the full width of the column. The btn-primary class is used to give the buttons a blue background color.

The col-md-3 and col-md-9 classes are used to create a two-column layout, with the left-hand side column taking up 3 out of 12 grid columns and the right-hand side column taking up 9 out of 12 grid columns. You can replace the content in the right-hand side column with your own content.
